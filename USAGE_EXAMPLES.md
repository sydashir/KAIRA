This guide provides practical examples of how to use KAIRA to generate different types of Spanish/Latin lyrics.

## Tips for Best Results

### 1. **Be Specific in Notes**
Instead of: "Make it good"
Try: "Need Caribbean vibes, summer romance theme, reference the beach"

### 2. **Match Energy to Genre**
- Dembow/Trap: High energy
- Bachata Urbana: Medium/Medium-Low
- Ballads: Low/Medium-Low

### 3. **Use Appropriate Slang Density**
- 0-3: Formal, mainstream appeal
- 4-6: Balanced, authentic but accessible
- 7-10: Heavy street slang, underground feel

### 4. **Leverage Phonetics**
Enable phonetics when:
- Working with difficult Spanish pronunciation
- Creating tracks for non-native speakers
- Using heavy regional slang

### 5. **Structure Selection**
- **Complete structures**: For full song generation
- **Freestyle**: For more creative, unconventional lyrics
- **Custom**: When you have a specific vision

### 6. **Language Mixing**
- **Spanglish**: Great for crossover appeal
- **Regional variations**: Adds authenticity
- **Primarily English with Spanish phrases**: Commercial appeal

## Workflow Suggestions

### For Songwriters:
1. Start with "Chorus" to get the main hook
2. Generate verses that complement the chorus theme
3. Add bridge for variation
4. Generate chanteo sections separately if needed

### For Producers:
1. Generate complete structure based on beat
2. Use QA log to understand cultural references
3. Adjust slang density to match target audience
4. Export JSON to keep metadata with your project

### For Learning Spanish:
1. Enable phonetics for every generation
2. Start with lower slang density (2-4)
3. Review QA log to understand cultural context
4. Gradually increase slang density as you learn

---

## Export and Usage

### TXT Format
Best for:
- Quick reference during recording
- Sharing with vocalists
- Printing for studio sessions

### JSON Format
Best for:
- Keeping complete project metadata
- Batch processing multiple generations
- Integration with other tools
- Archiving creative decisions

## Need Help?

If you're not getting the results you want:
1. Be more specific in the "Notes" field
2. Adjust slang density to match your vision
3. Try different genre combinations
4. Review the QA log to understand the AI's creative choices
5. Generate multiple variations and pick the best
